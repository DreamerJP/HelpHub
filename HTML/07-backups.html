Informações sobre Backups
  • O sistema realiza automaticamente um backup diário do banco de dados durante o primeiro login do dia.
  • São mantidos os backups dos últimos 14 dias no servidor, sendo os mais antigos substituídos pelos novos.
  • Cada backup é um arquivo completo do banco de dados, contendo todos os dados do sistema no momento em que foi gerado.
